# 📚 Estruturas de Dados

Repositório destinado às listas de exercícios da disciplina de **Estruturas de Dados**, do curso de **Ciência da Computação**.

---

## 🎯 Objetivo

Este repositório tem como objetivo armazenar as resoluções das listas de exercícios da disciplina, com foco na prática dos principais conceitos de Estruturas de Dados e no desenvolvimento da lógica de programação.

---

## 🛠️ Tecnologias Utilizadas

- 🐍 Python  
- 💻 Visual Studio Code  
- 🌿 Git  
- 🐙 GitHub  

---

## 📌 Conteúdo

Os exercícios abordam conceitos como:

- Listas
- Pilhas 
- Filas 
- Árvores 
- Algoritmos de busca
- Algoritmos de ordenação

---

## 📂 Organização do Repositório

O repositório está organizado em diretórios por lista de exercícios:

- Lista 1  
- Lista 2  
- Lista 3  
- Lista 4  
- ... (e assim por diante)

---

## 🎓 Finalidade

Projeto desenvolvido para fins acadêmicos, auxiliando no aprendizado da disciplina de Estruturas de Dados.
